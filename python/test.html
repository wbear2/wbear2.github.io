Getting Started

VIM常用命令

  1. array
  2. dtype
  3. reshape resize shape= 改变向量结构 ravel多维变一维，不是复制，值随源向量变化而变化 flatten多维变一维，复制，值不随源向量变化 transpose向量转置
  4. hstack 水平叠加 vstack 垂直叠加 concatenate 需要指定叠加方向 dstack 深度叠加，两个二维向量叠加成三维向量 column_stack 对一维向量叠加时，以列方式叠加；对二维向量叠加时，同hstack row_stack 对一维向量叠加时，以行方式叠加；对二维向量叠加时，同vstack
  5. hsplit vsplit split dsplit 拆分向量，类似上面的叠加
