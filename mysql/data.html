mysql install

install mysql 5.7.13 on win7

  • run cmd as administrator
  • execute command mysqld --install, install mysqld service
  • execute command mysqld --initialize, init mysqld, create data directory automatically
  • execute command net start mysql, start mysql service
  • notes: no need to modify any configurations or create any directory manually
